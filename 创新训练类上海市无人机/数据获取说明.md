# 上海市无人机项目数据获取说明

本次已优先下载公开可获得、且能直接用于前期空间分析和申报展示的数据。数据按类别存放在本文件夹下的各子目录中。

## 已下载数据

### 1. 基础地理数据

- `01_基础地理数据/shanghai-latest-free.gpkg.zip`
  - 来源：Geofabrik / OpenStreetMap 上海提取包
  - 用途：可作为项目的核心底图数据源
- `01_基础地理数据/shanghai_osm_gpkg/shanghai.gpkg`
  - 来源：上面压缩包解压结果
  - 用途：可直接在 ArcGIS / QGIS 中打开
  - 主要图层：
    - `gis_osm_adminareas_a_free`：行政边界面
    - `gis_osm_buildings_a_free`：建筑轮廓
    - `gis_osm_landuse_a_free`：土地利用
    - `gis_osm_roads_free`：道路网络
    - `gis_osm_pois_free` / `gis_osm_pois_a_free`：POI
    - `gis_osm_transport_free`：交通设施
    - `gis_osm_traffic_free`：交通相关点位
    - `gis_osm_water_a_free` / `gis_osm_waterways_free`：水体与河道
- `01_基础地理数据/shanghai.poly`
  - 来源：Geofabrik
  - 用途：上海范围裁剪边界文件
- `01_基础地理数据/geoBoundaries-CHN-ADM2.geojson`
  - 来源：geoBoundaries
  - 用途：全国 ADM2 行政区划边界，可作为上海区级边界备用源

### 2. 城市功能与需求数据

- `02_城市功能与需求数据/chn_ppp_2022_1km_UNadj.tif`
  - 来源：WorldPop
  - 含义：2022 年中国 1km 人口栅格
  - 用途：全国底版
- `02_城市功能与需求数据/上海市人口密度_WorldPop_2022_1km.tif`
  - 来源：由上面的 WorldPop 全国栅格裁剪得到
  - 含义：上海范围专用人口密度栅格
  - 用途：可直接用于上海人口密度、需求强度和供需匹配分析

说明：居住、商业、公共服务类 POI 已包含在 `shanghai.gpkg` 中，可在 `gis_osm_pois_free`、`gis_osm_pois_a_free`、`gis_osm_transport_free` 等图层内筛选。

### 3. 无人机配送现状参考材料

- `03_无人机配送现状数据/上海_公开可得起降相关点_OSM.gpkg`
- `03_无人机配送现状数据/上海_公开可得起降相关点_OSM.csv`

用途：作为公开可得的起降相关点代理层，便于前期识别机场、直升机起降点等低空设施相关位置。

说明：

- 这不是企业真实无人机配送站点运营库。
- 公开网页抓取到的企业介绍页已移除，避免误用。

### 4. 无人机技术参数参考材料

- `04_无人机技术参数数据/DJI_FlyCart_30_User_Manual_v1.2.pdf`
- `04_无人机技术参数数据/DJI_FlyCart_30_Quick_Start_Guide.pdf`
- `04_无人机技术参数数据/DJI_FlyCart_30_官方手册索引.csv`

用途：作为 FlyCart 30 的官方手册与索引资料，可用于后续提取续航、载荷、飞行速度、作业模式等参数。

说明：

- 原先抓到的网页版规格页不是可直接使用的数据文件，已移除。

### 5. 动态运行数据

- `06_动态运行数据/上海天气_OpenMeteo_7天.json`
  - 来源：Open-Meteo
  - 内容：上海未来 7 天逐小时天气、降水、风速等
  - 用途：作为天气约束的公开接口样例

## 适合你们当前阶段直接做的事情

1. 用 `shanghai.gpkg` 先完成上海基础底图、道路、建筑、土地利用和 POI 的整理。
2. 用 `chn_ppp_2022_1km_UNadj.tif` 裁剪上海范围，构造人口密度指标。
3. 在 `gis_osm_pois_free` 中筛选医院、消防、学校、警务、社区中心、避难相关点位作为应急设施近似数据。
4. 从 `DJI_FlyCart_30_User_Manual_v1.2.pdf`、`DJI_FlyCart_30_Quick_Start_Guide.pdf` 和 `DJI_FlyCart_30_官方手册索引.csv` 中提取典型无人机参数，先搭建约束模型原型。

## 说明

有些数据并非公开结构化下载即可获得，比如企业运营站点、禁飞区、实时交通、问卷访谈、建设成本台账等。这些内容已在 `待申请或待调研数据清单.md` 中单独列出。
